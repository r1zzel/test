Чтобы не забыть основное из Git:
git init 
    для создания нового репозитория на нашей локальной машине.
git remote 
    для подключения нашего локального репозитория к GitHub.
git branch 
    за переименование нашей ветки.
git add 
    а также за постановку и фиксацию наших изменений.git commit
git status 
    для проверки состояния наших файлов.
git push 
    за публикацию нашего кода на GitHub
